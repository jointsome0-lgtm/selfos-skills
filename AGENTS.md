# selfos-skills — Portable Agent Skills Catalog

The installable source of truth is `skills/<name>/`, following the open Agent Skills format. Each skill's `metadata.selfos.version` is canonical, and every change in that installable tree requires a strict semantic-version increase. Bundled dependency copies are generated from each composed skill's `BUNDLE.json` manifest, self-identify through a `GENERATED.md` marker, and retain their source skill's version, while the containing composed skill is bumped when its regenerated tree changes. Both host adapter versions are generated by separately summing the catalog's major, minor, and patch components; substantive aggregate-adapter changes require packaging bumps for every skill. Installers and native plugin systems discover those folders directly; this file is only a generated catalog and a fallback for agents that can read repository instructions but do not yet support skills.

When working inside this repository, read a matching skill's `SKILL.md` in full and resolve every relative path from that skill folder. Rows marked **explicit only** must not auto-activate from a topical match. This repository is public: use invented examples only and never commit personal data, credentials, private paths, or local agent state.
